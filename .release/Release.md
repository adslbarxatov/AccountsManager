_Изменения для v 1.34 от `20.01.2020`_:
- Устранена фатальная ошибка при работе с интерфейсом кассового плана;
- Устранена скрытая ошибка при работе с интерфейсом оплаты счетов;
- Исправлено выравнивание сумм в интерфейсе просмотра наблюдателей;
- Скрыты ошибки, связанные с работой «Корзины», проявляющиеся при некорректной системной дате (ввиду необязательности этого инструмента в текущей концепции);
- Устранена ошибка при добавлении записей в «Корзину» при неправильной системной дате: такие записи в корзину попадать не будут до исправления системной даты;
- Сортировка по номеру документа в наблюдателе видов расходов теперь выполняется корректно;
- Добавлено выделение суммовой строки в интерфейсе редактирования документов ЛБО;
- Текущая редактируемая строка в интерфейсе документа ЛБО теперь будет оставаться выделенной после изменения. Экспериментальная функция: если она будет работать корректно, мы сможем добавить её в остальные интерфейсы;
- В интерфейсе детализации ЛБО вместо кода вида работ в заголовке окна теперь отображается текстовая расшифровка вида работ;
- Исправлены подписи к некоторым элементам управления в интерфейсе детализации ЛБО;
- Оптимизирован общий интерфейс доступа к таблицам и справочникам;
- При запросе расшифровки из другого документа ЛБО теперь автоматически устанавливается фильтрация по текущему коду вида работ;
- В интерфейсе детализации ЛБО добавлено округление суммы до 100 рублей вверх и вниз, а также предпросмотр этой суммы;
- В интерфейсе детализации ЛБО добавлено отображение расшифровываемой суммы, переданной из интерфейса ЛБО (если сумма была указана ранее, но не была расшифрована);
- Длина наименования (комментария) расшифровки ЛБО увеличена до 150 символов;
- Прекращена поддержка версий базы данных 1.23 – 1.30;
- Неоплаченные и частично оплаченные периоды контрактов теперь помечаются в списке в интерфейсе оформления счёта словами `НЕ ОПЛ.` и `НЕПОЛН.` соответственно;
- Название детализации теперь предзаполняется по названию кода вида работ;
- Добавлена возможность изменения размера окна редактирования контрактов
